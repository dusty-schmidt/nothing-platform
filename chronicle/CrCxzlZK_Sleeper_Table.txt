#### What fields are available for player data?

The player data includes the following fields:

- **player_id**: Unique identifier for the player
- **hashtag**: Display hashtag (e.g., "#TomBrady-NFL-NE-12")
- **first_name**: Player's first name
- **last_name**: Player's last name
- **search_full_name**: Full name for searching purposes
- **search_first_name**: First name for searching
- **search_last_name**: Last name for searching
- **position**: Player's position (e.g., QB, RB, WR)

- **team**: Team abbreviation (e.g., NE, PIT, ARI)

- **injury_status**: Injury status (e.g., "Out", "Probable", or empty)

- **height**: Player's height
- **weight**: Player's weight
- **age**: Player's age
- **college**: College attended
- **years_exp**: Years of experience
- **birth_country**: Country of birth
- **depth_chart_position**: Position on depth chart
- **depth_chart_order**: Order on depth chart
- **practice_participation**: Practice participation status

#### What are the available API endpoints?

The Sleeper API provides several main categories of endpoints:

- **User Endpoints**: Fetch user objects by username or user_id (`GET https://api.sleeper.app/v1/user/<username>` or `GET https://api.sleeper.app/v1/user/<user_id>`)

- **League Endpoints**:
  - Get all leagues for a user: `GET https://api.sleeper.app/v1/user/<user_id>/leagues/<sport>/<season>`
  - Get a specific league: `GET https://api.sleeper.app/v1/league/<league_id>`
  - Get rosters: `GET https://api.sleeper.app/v1/league/<league_id>/rosters`
  - Get users: `GET https://api.sleeper.app/v1/league/<league_id>/users`
  - Get matchups: `GET https://api.sleeper.app/v1/league/<league_id>/matchups/<week>`
  - Get transactions: `GET https://api.sleeper.app/v1/league/<league_id>/transactions/<round>`
  - Get traded picks: `GET https://api.sleeper.app/v1/league/<league_id>/traded_picks`
  - Get winners/losers bracket: `GET https://api.sleeper.app/v1/league/<league_id>/winners_bracket` and `GET https://api.sleeper.app/v1/league/<league_id>/losers_bracket`

- **Draft Endpoints**:
  - Get all drafts for a user: `GET https://api.sleeper.app/v1/user/<user_id>/drafts/<sport>/<season>`
  - Get all drafts for a league: `GET https://api.sleeper.app/v1/league/<league_id>/drafts`
  - Get a specific draft: `GET https://api.sleeper.app/v1/draft/<draft_id>`
  - Get all picks in a draft: `GET https://api.sleeper.app/v1/draft/<draft_id>/picks`
  - Get traded picks in a draft: `GET https://api.sleeper.app/v1/draft/<draft_id>/traded_picks`

- **Player Endpoints**:
  - Fetch all players: `GET https://api.sleeper.app/v1/players/nfl`
  - Get trending players: `GET https://api.sleeper.app/v1/players/<sport>/trending/<type>`

- **State Endpoint**: Get NFL state: `GET https://api.sleeper.app/v1/state/nfl`
