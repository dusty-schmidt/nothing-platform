### document_query
Read and query local or remote documents (HTML, PDF, Office, text).

**Args:** `document` (url, path, or list of either), `queries` (list of strings — omit to return full content)
- Local: full path required, file:// prefix optional
- Remote: http:// or https:// prefix required
