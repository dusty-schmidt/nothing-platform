### skills_tool
On-demand capability packages. Index listed below under Available skills.

- `skills_tool:list` — list all skills with metadata
- `skills_tool:load skill_name` — load full SKILL.md before using
- Execute skill scripts via `code_execution_tool` terminal runtime

Always load before use. Reload if scrolled out of history.
